## Стартовые условия

Когда ты запускаешь игру в первый раз, тебе выдаётся:

| Параметр | Значение |
|---|---|
| Деньги | **200 монет** |
| Транспорт | 1 пеший курьер (бесплатно, выдаётся автоматически) |
| Первый сотрудник | Новичок **«Alex»** (нанят автоматически, без оплаты найма) |
| Открытый район | **Жилой** (Residential) |

Больше ничего нет. Гараж нужно построить самому, прежде чем купить хоть один автомобиль. Единственная доступная работа - брать заказы в Жилом районе и доставлять их пешком.

## Игровой тик и время

**1 тик = 1 секунда реального времени.**

Всё в игре измеряется в тиках:

- Каждые **15 тиков** (= 15 секунд) во всех разблокированных районах появляются новые заказы.
- Загрузка транспорта занимает **10 тиков** (без грузчика) или **2 тика** (с грузчиком).
- У каждого заказа есть дедлайн в тиках - если дедлайн истёк до того, как заказ принят, он аннулируется.
- Зарплата сотрудников считается за **неделю** (расчётные периоды встроены в тик-цикл).

## Деньги и опыт

**Монеты** - единственная валюта в игре. Тратятся на найм, транспорт, здания.

**Опыт (XP)** - начисляется за каждую выполненную доставку:
```
XP = Награда за заказ × 0.1
```

Примеры:
- Жилой заказ (30 монет) -> **3 XP**
- Деловой заказ (150 монет) -> **15 XP**
- Промышленный заказ (300 монет) -> **30 XP**
- Межгородской заказ (800 монет) -> **80 XP**

Навык «Переговоры» увеличивает и монеты, и XP, так как XP считается от итоговой награды.

## Уровень игрока и навыки

### Формула уровня

```
Уровень = 1 + √(Опыт × 0.1)
```

| Уровень | Требуется XP | Что открывается |
|---|---|---|
| 1 | 0 | Жилой район (старт) |
| 2 | 100 XP | +1 очко навыков |
| 3 | 400 XP | Деловой район + очко навыков |
| 4 | 900 XP | +1 очко навыков |
| 5 | 1600 XP | +1 очко навыков |
| 7 | - | Промышленный район |
| 15 | - | Межгородской район |

За каждый новый уровень выдаётся **1 очко навыков**. Очки тратятся в меню прокачки.

---

### Навыки игрока

Три навыка, каждый прокачивается за очко навыков:

#### Выносливость (Stamina)
- Увеличивает запас сил пешего курьера
- Прирост: **+20 очков за уровень**
- Начальное значение: 100
- Максимум: **200** (уровень 5 навыка)
- Нужен для работы в режиме пешего курьера - без прокачки быстро устаёт на дальних заказах.

#### Скорость бега (Run Speed)
- Увеличивает скорость пешего курьера
- Прирост: **+0.1× за уровень**
- Начальная скорость: 0.5×
- Максимум: **3.0×** (25 уровней навыка)
- На максимуме пеший курьер движется со скоростью 3× - быстрее, чем Газель (0.8×), но всё ещё медленнее самоката (1.5×) при нормальной прокачке.

#### Переговоры (Negotiation)
- Увеличивает денежное вознаграждение за каждый заказ
- Прирост: **+5% к награде за каждый уровень**
- Максимум: **уровень 10** (+50% итого, то есть ×1.5 к любой награде)
- Применяется ко всем заказам всей компании, не только к курьеру.

Формула награды с навыком:
```
Итоговая награда = Базовая награда × (1 + Уровень_переговоров × 0.05)
```

На максимуме: межгородской заказ 800 монет -> **1200 монет**.

## Сотрудники

Сотрудники нанимаются в разделе «Персонал». У каждого есть уровень опыта (1–10) и надёжность (0–0.99). Надёжность растёт: **+0.01 за каждую выполненную доставку**, до максимума 0.99.

**Стоимость найма = Зарплата × 2 (авансом за 2 недели)**

Транспорт покупается в разделе «Флот». Для покупки любого транспорта (кроме пешего курьера) нужен построенный **Гараж** со свободным слотом.

### Состояния транспорта

| Статус | Что означает |
|---|---|
| **IDLE** | Свободен, готов к заказу |

| **REFUELING** | На заправке |

## Заказы

### Жизненный цикл заказа

```
PENDING -> ASSIGNED -> IN_PROGRESS -> COMPLETED
                                  -> FAILED (истёк дедлайн пока PENDING)
```

| Статус | Что происходит |
|---|---|
| **PENDING** | Заказ создан, ждёт назначения. Если дедлайн истекает - автоматически переходит в FAILED |
| **ASSIGNED** | Транспорт и водитель назначены, загрузка началась |
| **IN_PROGRESS** | Транспорт в пути, обратного пути нет - дедлайн не влияет |
| **COMPLETED** | Доставлено. Игрок получает монеты и XP |
| **FAILED** | Дедлайн истёк пока заказ был PENDING. Никакой награды |

**Важно:** заказ в статусе IN_PROGRESS нельзя отменить и нельзя провалить - он всегда будет доставлен.

### Генерация заказов

- Каждые **15 тиков** появляется новая пачка заказов во всех разблокированных районах.
- Тип груза определяется районом.
- Количество появляющихся заказов зависит от наличия Сортировочного центра.

### Награда с бонусом переговоров

```
Итог = Базовая награда × (1 + Навык_переговоров × 0.05)
XP = Итог × 0.1
```

## Маршруты и время доставки

### Расстояния между районами

| Откуда | Куда | Расстояние |
|---|---|---|
| Жилой | Деловой | 3 км |
| Жилой | Промышленный | 5 км |
| Деловой | Промышленный | 4 км |
| Промышленный | Межгород | **50 км** |

### Время доставки

```
Время (тики) = Расстояние / Скорость транспорта
              (но не меньше минимального по типу груза)
```

Минимальное время:
- Лёгкий груз -> **60 тиков**
- Средний груз -> **120 тиков**
- Тяжёлый груз -> **180 тиков**

**Примеры:**
- Пеший курьер (0.5×), 3 км, лёгкий груз: 3/0.5 = 6 -> но минимум 60 -> **60 тиков**
- Самокат (1.5×), 3 км, лёгкий груз: 3/1.5 = 2 -> но минимум 60 -> **60 тиков**
- Фура (0.6×), 50 км, тяжёлый груз: 50/0.6 = 83 -> но минимум 180 -> **180 тиков**

### Ограничения маршрутов

- **Фура (Semi-Trailer)** - не заезжает в Жилой и Деловой районы (высокий трафик).
- **Рефрижератор и Топливозаправщик** - ходят только по грузовым маршрутам (Промышленный, Межгород).

## Здания и инфраструктура

Здания строятся через раздел «Здания» в выбранном районе. Каждое здание можно улучшать.

**Формула стоимости улучшения:**
```
Стоимость улучшения = Базовая цена здания × Текущий уровень
```

---

### Гараж (Garage)

| Параметр | Значение |
|---|---|
| Цена постройки | **2 000 монет** |
| Базовая ёмкость | **3 машины** |
| Прирост за уровень | **+2 машины** |
| Максимальный уровень | **5** (итого 11 машин) |
| Стоимость улучшения | 2 000 × текущий уровень |

Пример стоимостей улучшений:
- Уровень 1 -> 2: 2 000 монет
- Уровень 2 -> 3: 4 000 монет
- Уровень 3 -> 4: 6 000 монет
- Уровень 4 -> 5: 8 000 монет

**Без гаража нельзя купить ни одну машину.** Это первое здание, которое нужно построить. Когда все слоты заняты - нужно улучшать гараж или продавать старый транспорт.

---

### Сортировочный центр (Sorting Center)

| Параметр | Значение |
|---|---|
| Цена постройки | **10 000 монет** |
| Базовые слоты | **5 заказов** |
| Прирост за уровень | **+5 слотов** |
| Максимальный уровень | **3** (итого 15 заказов) |
| Стоимость улучшения | 10 000 × текущий уровень |

Объединяет мелкие посылки для пакетного вывоза. При высоком потоке заказов Сортировочный центр резко повышает эффективность - вместо трёх отдельных рейсов один грузовик забирает сразу пакет.

---

### Заправочная станция (Gas Station)

| Параметр | Значение |
|---|---|
| Цена постройки | **25 000 монет** |
| Скидка на топливо | **20% за уровень** |
| Максимальный уровень | **3** (итого скидка 60%) |
| Стоимость улучшения | 25 000 × текущий уровень |

При наличии Gas Station (уровень 3):
- Ларгус: 20 монет -> **8 монет** за доставку
- Газель: 40 монет -> **16 монет**
- Фура: 100 монет -> **40 монет**
- Спецтранспорт: 50 монет -> **20 монет**

Также на Заправочной станции ремонтируется сломанный транспорт. При большом флоте окупается очень быстро.

---

### Ферма (Farm)

| Параметр | Значение |
|---|---|
| Цена постройки | **50 000 монет** |
| Производство | **1 зерно/тик × уровень** |
| Максимальный уровень | **5** (до 5 зерна/тик) |
| Стоимость улучшения | 50 000 × текущий уровень |

Ферма автоматически производит зерно каждый тик. Зерно - первое звено в цепочке поставок хлеба. Без фермы цепочку не запустить.

---

### Пекарня (Bakery)

| Параметр | Значение |
|---|---|
| Цена постройки | **40 000 монет** |
| Выход за партию | **3 хлеба × уровень** |
| Максимальный уровень | **5** (до 15 хлебов за партию) |
| Стоимость улучшения | 40 000 × текущий уровень |

Перерабатывает муку в хлеб. Хлеб продаётся через заказы с более высокой маржой, чем сырьё. Пекарня нужна для завершения производственной цепочки.

## Цепочка поставок

Помимо обычных заказов на доставку, в игре есть производственная цепочка. Она требует специальных зданий и транспорта.

```
Ферма              Мельница           Пекарня           Покупатель
(Зерно) ──[транспорт]──> (Мука) ──[транспорт]──> (Хлеб) ──[транспорт]──>
```

### Шаги цепочки

1. **Ферма -> Мельница:** везём GRAIN. Подходят Ларгус, Газель, Фура.
2. **Мельница -> Пекарня:** везём FLOUR. Те же транспорты.
3. **Пекарня -> Покупатель:** везём BREAD. Дополнительно могут везти Пеший курьер и Самокат.

Каждое звено - это отдельный заказ с отдельным транспортом. Для запуска полной цепочки нужно:
- Построить Ферму и Пекарню.
- Иметь минимум 3 свободных транспортных средства с нужными типами груза.
- Создать цепочку через кнопку «Создать цепочку поставок» в меню.

## Авто-диспетчеризация

При наличии хотя бы одного нанятого **Диспетчера** включается автоматический режим.

### Как работает каждый тик:
1. Диспетчер ищет все заказы в статусе **PENDING**.
2. Для каждого заказа подбирает **свободный транспорт** подходящего типа груза.
3. Назначает **первого доступного водителя** (Опытный водитель или Новичок).
4. Запускает доставку - заказ переходит в **ASSIGNED**, начинается загрузка.

### Важные ограничения:
- Если нет свободного транспорта нужного типа - заказ остаётся PENDING.
- Если нет свободного водителя - транспорт не отправляется.
- Диспетчер работает по всем районам одновременно.

Без диспетчера каждый заказ нужно назначать вручную в разделе «Заказы».

## Система усталости

Активна **только на межгородских маршрутах** (район Межгород, уровень 15+).

| Параметр | Значение |
|---|---|
| Максимальная усталость | 100 очков |
| Нарастание | +10 очков в час доставки |
| Порог отдыха | 80 очков - водитель уходит на отдых |
| Восстановление | Автоматически когда водитель не в рейсе |

При усталости 80+ водитель недоступен для новых заказов - нужно ждать восстановления или чередовать водителей. На коротких городских маршрутах усталость не влияет.
